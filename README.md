# ESP32-CRT-Console

An ESP32-based composite video game console using the [esp_8_bit](https://github.com/rossumur/esp_8_bit) project. Outputs video over composite to a CRT display and emulates NES, Atari 2600, and Atari 7800 games directly from the ESP32.

## Contents

- `esp_8_bit/` — ESP32 composite video emulator source
- `IMG_5444.HEIC` – `IMG_5452.HEIC` — Build photos of the physical console

## Hardware

- ESP32 dev board
- Composite CRT display
- 3.5mm composite cable (wired to ESP32 DAC pin)
